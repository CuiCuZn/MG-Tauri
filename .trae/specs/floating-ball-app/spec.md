# 桌面悬浮球应用规格说明书

## 一、需求概述

创建一个桌面悬浮球应用，类似于搜狗输入法的悬浮球功能。应用包含一个始终置顶的悬浮球，用户点击悬浮球上的按钮可以弹出内容窗口，每个按钮对应打开百度搜索页面。

## 二、变更内容

### 2.1 窗口配置变更
- 新增悬浮球窗口：透明背景、圆形、始终置顶、无边框
- 新增内容展示窗口：用于显示百度搜索页面

### 2.2 UI 组件
- 悬浮球组件：包含 5 个功能按钮
- 搜索结果显示组件：WebView 加载百度搜索页面

### 2.3 窗口通信
- 悬浮球与内容窗口之间的显示/隐藏通信

## 三、影响范围

### 3.1 受影响的规格
- 窗口管理系统
- 插件系统

### 3.2 受影响的代码
- `src-tauri/tauri.conf.json` - 窗口配置
- `src-tauri/src/lib.rs` - Tauri 命令
- `src/App.vue` - 悬浮球 UI
- `src/components/` - 新增组件

## 四、新增需求

### 需求：悬浮球功能
系统 SHALL 提供一个始终置顶的桌面悬浮球，用户可以通过它快速访问预设功能。

#### 场景：悬浮球显示
- **GIVEN** 应用启动
- **THEN** 悬浮球窗口显示在桌面右下角区域
- **AND** 悬浮球始终置顶于其他窗口

#### 场景：点击悬浮球按钮
- **WHEN** 用户点击悬浮球上的功能按钮
- **THEN** 弹出内容展示窗口
- **AND** 窗口加载百度搜索页面

#### 场景：关闭内容窗口
- **WHEN** 用户关闭内容展示窗口
- **THEN** 仅关闭该内容窗口，悬浮球保持显示

### 需求：五个功能按钮
系统 SHALL 提供五个预设的功能按钮，每个按钮对应不同的百度搜索功能。

| 按钮 | 功能 | 搜索 URL |
|------|------|----------|
| 按钮1 | 网页搜索 | https://www.baidu.com/s?wd= |
| 按钮2 | 图片搜索 | https://image.baidu.com/search/index?tn=baiduimage |
| 按钮3 | 视频搜索 | https://v.baidu.com/ |
| 按钮4 | 新闻搜索 | https://news.baidu.com/ |
| 按钮5 | 天气查询 | https://weather.baidu.com/ |

#### 场景：按钮点击响应
- **WHEN** 用户点击任一按钮
- **THEN** 内容窗口打开并加载对应百度搜索页面

## 五、修改需求

### 需求：窗口配置
原配置：单个窗口显示
修改为：
- 悬浮球窗口：50x50 像素、圆形、透明背景、始终置顶
- 内容窗口：800x600 像素、可调整大小

## 六、移除需求

无

## 七、技术实现要点

### 7.1 Tauri 配置
- 使用 `decorations: false` 移除窗口装饰
- 使用 `alwaysOnTop: true` 实现悬浮球置顶
- 使用 `transparent: true` 实现透明背景
- 使用 `visible` 控制窗口显示/隐藏

### 7.2 前端实现
- 使用 Vue 3 Composition API
- 使用 Tauri API 控制窗口

### 7.3 Rust 后端
- 提供窗口控制命令
- 处理前端请求
